# java-filmorate
Template repository for Filmorate project.
![]([https://github.com/Ilux-i/java-filmorate/blob/controllers-films-users/src/main/resources/scheme.png?raw=true](https://github.com/Ilux-i/java-filmorate/blob/add-database/src/main/resources/scheme.png))

В папке resources:
    schema.sql - Описание таблиц, хранящих сущности.
    data.sql - Описание данных для неизменяемых таблиц, которые автоматически заносятся в них.
